Scope and Limit

Overview

In terms of limits, as we only have 6 weeks to work on this assignment, the deliverables can be very limited to Figma prototype screens instead of a fully developed application that can work on mobile devices. Taking other factors such as economic and time constraints, we would at the least need a few weeks of working on the project to progress further.

Project Objective:

Create a mobile application that allows the fast and efficient delivery of various items found in stores located within popular shopping centres

Resources:

Team leader(1), Development team(3), Development leader(1), Software tester(1)

Deliverables:

A list of Figma prototype screens that will give a basic overview of the application in terms of visuals, Full PDF Report outlining information about the project, GitHub Pages website that will also outline the same information in the report.

Out of scope:

A fully furnished application that will be available on iOS and Android devices, a separate website that allows stores to manage their statistics
